# Jarvis-GPT 🤖

An AI-powered desktop voice assistant built using Python.

## ✨ Features

- Voice command recognition
- AI chatbot integration
- GUI interface
- Weather updates
- System controls
- Music playback
- Web search support

## 🛠️ Tech Stack

- Python
- CustomTkinter
- SpeechRecognition
- PyAudio
- Pillow (PIL)
- Requests
- PyAutoGUI
- Google Gemini / OpenAI API

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/Jarvis-GPT.git
cd Jarvis-GPT
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

## ▶️ Run the Assistant

```bash
python chatbot1.py
```
